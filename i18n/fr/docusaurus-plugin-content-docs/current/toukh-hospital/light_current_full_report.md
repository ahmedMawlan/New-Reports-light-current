Le travail comprend les systèmes suivants:

- Système d'alarme incendie
- Système de contrôle d'accès

- Système de sonorisation 
- Système de télécommunications 
- Système d'appel infirmier 
- Système satellitaire

## 🗺️ 4. Schémas 

### 🔹 schéma de vidéosurveillance

    Caméra → Cat6 → POE Switch → NVR → Routeur → Internet

### 🔹 Schéma de boucle d'alarme incendie

    Panneau → Fumée → Chaleur → MCP → Sondeur → Retour au panneau

### 🔹 diagramme de contrôle d'accès

    Lecteur de cartes → Contrôleur → Serrure magnétique → Bouton de sortie → Alimentation électrique

### 🔹 Diagramme de réseaux de données

    Internet → Pare-feu → Routeur → core Switch → Manette sans fil → Points d'accès → Serveurs → Ordinateur

### 🔹 Schéma de sonorisation

    Microphone → mixeur / Amplificateur → haut-parleurs → Câblage / Réseau → Contrôle du volume

### 🔹 Diagramme de télécommunications

    PSTN / ISP  → PBX → Routeur → Switch → Téléphones analogiques / IP

### 🔹 Schéma d'appel infirmier

    Bouton d'appel → Cordon de tirage → FJB → Panneau de la station infirmière → Écran

### 🔹 Diagramme de satellite

    Antenne  → LNB → Multi Switch → Amplificateur → Écran

### 🔹 Queuing Diagram

    Distributeur de Ticket  → Ticket → Écrans → Unité d'appel de comptoir
