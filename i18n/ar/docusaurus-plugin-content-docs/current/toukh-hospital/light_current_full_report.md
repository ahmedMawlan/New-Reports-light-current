## 🗺️ 4. مخطط التوصيل 

### 🔹 مخطط نظام الكاميرات

    Camera → Cat6 → POE Switch → NVR → Router → Internet

### 🔹 مخطط نظام الحريق

    Panel → Smoke → Heat → MCP → Sounder → Back to Panel

### 🔹 مخطط نظام التحكم في الدخول 

    Card Reader → Controller → Magnetic Lock → Exit Button → Power Supply

### 🔹 مخطط نظام شبكة البيانات

    Internet → Firewall → Router → core Switch → Wireless Controller → Access Points → Servers → PC

### 🔹 مخطط نظام الصوت

    Microphone → Mixer / Amplifier → Speakers → Cabling / Network → Volume Control

### 🔹 مخطط نظام الاتصالات

    PSTN / ISP  → PBX → Router → Switch → Phones

### 🔹 مخطط نظام الاستدعاء للممرضات

    Call Button  → Pull Cord → FJB → Nurse Station Panel → Screen

### 🔹 مخطط نظام التلفاز

    Antenna  → LNB → Multi Switch → Amplifier → Screen

### 🔹 مخطط نظام الاصطفاف

    Ticket Dispenser  → Ticket → Screens → Counter Call Unit
